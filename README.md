# Sakila Video Rental Analytics

End-to-end data analytics project built using MySQL and Power BI.

The project analyzes store performance, customer behavior, film popularity, and churn risk using the Sakila database.

## Tech Stack

- MySQL
- Power BI
- DAX
- Data Cleaning
- Customer Segmentation
- Churn Analysis

## Business KPIs

- Total Revenue
- Rentals per Store
- Average Ticket
- Top Customers
- Revenue Trends
- Churn Risk

## Churn Prediction Logic

A churn score was built using:
- Recency
- Frequency
- Monetary behavior

Customers were classified into:
- Low Risk
- Medium Risk
- High Risk

## Key Insights

- Store 1 generated the highest revenue
- VIP customers contributed disproportionately to sales
- Majority of rentals happened in July/August
- Films in animation and action categories dominated rentals
- High churn risk customers showed low rental frequency and recency
